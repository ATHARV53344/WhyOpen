# 🚀 WhyOpen – Mindful Browsing Chrome Extension

**WhyOpen** is a productivity-focused Chrome extension that helps you stay focused by asking a simple but powerful question:

> **"Why did you open this tab?"**

It reminds you of your intent and prevents mindless browsing.

---

## ✨ Features

### 🧠 Core Features

* Ask users **why they opened a tab**
* Save purpose per tab
* Show reminders when revisiting tabs

### ⚡ AI Purpose Matching (Smart Feature)

* Detects mismatch between your goal and current page
* Warns when you're likely distracted

Example:

> You opened this tab to *Study React*
> This page may distract you.

---

### 📊 Weekly Productivity Report

* Total tabs opened
* Purposeful vs distracted tabs
* Focus score calculation
* Top visited websites
* History of reasons

---

### 🔔 Notifications

* Real-time focus reminders
* Distraction alerts

## ⚙️ How to Install (Developer Mode)

Since this extension is not published yet, you can run it locally.

### Step 1: Download the Project

Clone or download this repository:

```
git clone <your-repo-link>
```

or download ZIP and extract it.

---

### Step 2: Open Chrome Extensions

Go to:

```
chrome://extensions
```

---

### Step 3: Enable Developer Mode

Turn ON **Developer Mode** (top right corner)

---

### Step 4: Load the Extension

Click:

```
Load unpacked
```

Then select the project folder.

---

### Step 5: Done 🎉

Your extension is now running!

---

## 🧪 How to Use

1. Open a new tab
2. Click the extension icon
3. Enter your reason
4. Browse normally

You will:

* Get reminders of your purpose
* Be warned if you get distracted
* See analytics in dashboard

---

## 📈 Example Use Case

```
Reason: Study Operating Systems
```

Later:

```
Opening YouTube memes...
```
